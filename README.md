# 🛒 电商选品仓库

> 用于网络选品卖货交易的资料管理和运营分析

## 📁 文件结构

```
├── README.md              # 本文件，仓库说明
├── products/              # 商品选品记录
│   └── 选品记录.md        # 选品信息汇总表
├── analysis/             # 爆款分析
│   └── 爆款分析.md        # 爆款商品分析报告
├── data/                 # 销售数据
│   └── 销售数据.md        # 日常销售记录
└── notes/                # 运营笔记
    └── 运营笔记.md        # 日常运营记录
```

## 🚀 快速开始

1. **记录新品** → 在 `products/选品记录.md` 中添加
2. **分析爆款** → 在 `analysis/爆款分析.md` 中记录
3. **更新销售** → 在 `data/销售数据.md` 中登记
4. **日常总结** → 在 `notes/运营笔记.md` 中书写
